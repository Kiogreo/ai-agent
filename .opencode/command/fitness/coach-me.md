# Coach Me Command

## Overview
Provides personalized coaching with motivational support and kink-aware encouragement for bear aesthetic development.

## Process
1. Assess current state and needs
2. Provide motivational feedback
3. Offer specific guidance and recommendations
4. Include bear aesthetic encouragement

## Usage Examples
- `/coach-me` - General coaching session
- `/coach-me focus_area=motivation` - Motivation-focused coaching
- `/coach-me quick_check=true` - Quick encouragement boost

## Features
- Kink-aware motivational messaging
- Bear aesthetic focus
- Personalized recommendations
- Progress celebration
