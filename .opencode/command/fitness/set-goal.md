# Set Goal Command

## Overview
Helps establish or modify fitness goals aligned with bear aesthetic development priorities.

## Process
1. Analyze current progress and capabilities
2. Set realistic, measurable goals
3. Create action plan for goal achievement
4. Establish tracking and check-in schedule

## Usage Examples
- `/set-goal goal_type=strength specific_target="bench press 225lbs"`
- `/set-goal goal_type=aesthetic timeframe=quarter`

## Goal Types
- **strength**: Specific lift targets
- **aesthetic**: Body composition goals
- **consistency**: Training frequency targets

## Validation
- Goals must align with bear aesthetic priorities
- Realistic based on current progress
- Measurable and time-bound
