# Progress Tracking Workflow

## Data Collection Phase
1. Retrieve Liftosaur workout data
2. Gather current measurements
3. Assess training consistency
4. Review nutrition adherence

## Analysis Phase
1. Calculate volume progression
2. Analyze exercise balance
3. Evaluate recovery patterns
4. Identify strength improvements

## Reporting Phase
1. Generate progress summary
2. Highlight achievements
3. Identify areas for improvement
4. Provide next milestone targets

## Recommendation Phase
1. Suggest training adjustments
2. Recommend recovery strategies
3. Update nutrition guidance
4. Set new short-term goals

## Follow-up Phase
1. Schedule progress check-in
2. Update tracking parameters
3. Adjust monitoring frequency
4. Celebrate milestones achieved
